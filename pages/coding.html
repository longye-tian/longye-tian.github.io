EconRL: Reinforcement Learning for Economics

A comprehensive Python library implementing state-of-the-art reinforcement learning algorithms specifically designed for economic modeling. Features include specialized environments for job search, optimal taxation, and dynamic pricing problems. Used by researchers at 5+ universities.

example.py Python 
from econrl import JobSearchEnv, DQNAgent

# Initialize environment and agent
env = JobSearchEnv(wage_dist='lognormal', search_cost=5)
agent = DQNAgent(state_dim=env.state_dim, action_dim=2)

# Train the agent
agent.train(env, episodes=1000)
optimal_policy = agent.get_policy()
